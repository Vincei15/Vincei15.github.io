Admin Bulk Upload

Upload a bulk product data file in JSON, or TXT format.



Example File Format

Ensure your file follows one of these formats:

JSON Format

   [
        {
                                "name": "Jordan 1's",
                            "price": 710.00,
                            "stock": 10
                },

        {
                                "name": "Jordan 2's",
                                "price": 210.00,
                                "stock": 15
                }
    ]

TXT Format

    Jordan 1's | 710.00 | 10
    Jordan 2's | 210.00 | 15

Only JSON, and TXT files are accepted. Ensure correct formatting before uploading.
